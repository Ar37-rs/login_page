# Fullstack_adya_roby

Clone repo:

```bash
git clone https://github.com/Ar37-rs/fullstack_adya_roby.git
cd fullstack_adya_roby
```

jalankan BE:

```bash
cd fullstack_go_backend
go mod tidy
go run main.go
```

jalankan FE:

```bash
cd fullstack_js_frontend
npm install
npm start
```

kemudian buka http://localhost:3000
